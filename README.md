# Phantom-DoS
Phantom DoS is a tool designed for DoS attacks. (Denial of Service)
# How does it work?
Simple;
Download the "PHANTOM DoS.exe" file, enter the website url and referrer url.
Once done, the website will start being flooded with requests.
